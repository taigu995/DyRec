# DyRec - 抖音直播录制管理工具

## 快速开始

### 前置要求

1. **Node.js 20+** - [下载地址](https://nodejs.org/zh-cn)
2. **FFmpeg** (可选，录制功能需要) - [下载地址](https://ffmpeg.org/download.html)

### 启动步骤

1. 双击运行 `start.bat`
2. 浏览器会自动打开 `http://localhost:5000`
3. 开始使用！

### FFmpeg 安装

#### 方法一：放到指定目录
1. 下载 FFmpeg (Windows 版本)
2. 解压后将 `ffmpeg.exe` 放到本目录下的 `ffmpeg` 文件夹中

#### 方法二：添加到系统 PATH
1. 下载 FFmpeg 并解压
2. 将解压目录添加到系统环境变量 PATH 中

### 功能说明

- **直播间管理**：添加抖音直播间，自动检测开播状态
- **直播预览**：手机端视角预览，实时弹幕滚动，礼物特效显示
- **录制功能**：
  - 合成录制：画面 + 弹幕 + 礼物效果（WebM 格式）
  - 原始流录制：最高画质录制（TS/FLV 格式）
- **环境检测**：一键检测并安装前置依赖

### 注意事项

- 首次启动可能需要等待几秒钟
- 录制功能需要 FFmpeg
- 合成录制功能在浏览器中运行，需要 Chrome/Edge 浏览器
- 录制文件保存在 `recordings` 目录

### 常见问题

**Q: 启动后浏览器没有自动打开？**
A: 请手动访问 http://localhost:5000

**Q: 提示找不到 FFmpeg？**
A: 请按照上面的 FFmpeg 安装步骤操作

**Q: 如何停止服务？**
A: 在命令行窗口按 Ctrl+C
